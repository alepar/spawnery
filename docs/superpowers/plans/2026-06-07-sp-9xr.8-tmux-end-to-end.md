## Task 2: Idle-reap tmux relays

**Files:** `internal/node/tmuxrelay.go`, `internal/node/attach.go`, `internal/node/tmuxrelay_test.go`

- [ ] **Step 1: Add activity tracking to tmuxRelay**

In `internal/node/tmuxrelay.go`, add `lastActivity time.Time` (guarded by `r.mu`) and helpers; mark activity on `attach`, `fromClient`, and each PTY→client `send`:

```go
// in the struct: lastActivity time.Time  (set to time.Now() in newTmuxRelay)
func (r *tmuxRelay) markActive() { r.mu.Lock(); r.lastActivity = time.Now(); r.mu.Unlock() }
func (r *tmuxRelay) lastActive() time.Time { r.mu.Lock(); defer r.mu.Unlock(); return r.lastActivity }
func (r *tmuxRelay) attached() bool { r.mu.Lock(); defer r.mu.Unlock(); return len(r.clients) > 0 }
```

Call `r.markActive()` in `attach` (start), in `fromClient` (on input), and in the reader goroutine right before/after `r.send(...)`. (`import "time"`.)
