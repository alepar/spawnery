# sp-9xr.3.3 — Image catalog registration (node advertises binaries → CP upserts) Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** Populate the `agent_images` catalog automatically: a node advertises the binaries its image ships in its `Register` message, and the CP upserts each `(image → binaries)` into the catalog on registration (idempotent across reconnects).

**Architecture:** Add `AgentBinaries` to the node `Config`, thread it into the `Register` message (via an extracted, testable `registerMessage` helper), and wire `AGENT_BINARIES` env in `cmd/spawnlet`. On the CP, add an `upsertAgentCatalog` helper called from the `Register` handler that writes each advertised image + its binaries into the store. Depends on `sp-9xr.3.2` (the `Register.binaries` proto field, already merged).

**Tech Stack:** Go (`spawnery`), Connect bidi node stream, Bun store, stdlib `testing`.

**Run Go with:** `export PATH="/var/home/alepar/sdk/go1.26.0/bin:$PATH"`.

```go
package cp

import (
	"context"
	"testing"
)

func TestUpsertAgentCatalog(t *testing.T) {
	s, _, _ := newTestServer(t)
	ctx := context.Background()

	// First registration records the image + its binaries.
	s.upsertAgentCatalog(ctx, []string{"ghcr.io/acme/goose:1"}, []string{"goose", "opencode"})
	if _, err := s.st.AgentImages().Get(ctx, "ghcr.io/acme/goose:1"); err != nil {
		t.Fatalf("image not recorded: %v", err)
	}
	bins, err := s.st.AgentImages().Binaries(ctx, "ghcr.io/acme/goose:1")
	if err != nil || len(bins) != 2 || bins[0] != "goose" || bins[1] != "opencode" {
		t.Fatalf("binaries = %v err = %v", bins, err)
	}

	// Reconnect replaces the binary set (idempotent), catalog row persists.
	s.upsertAgentCatalog(ctx, []string{"ghcr.io/acme/goose:1"}, []string{"claude-code"})
	bins, _ = s.st.AgentImages().Binaries(ctx, "ghcr.io/acme/goose:1")
	if len(bins) != 1 || bins[0] != "claude-code" {
		t.Fatalf("reconnect should replace binaries, got %v", bins)
	}

	// Empty image names are skipped (no spurious catalog row).
	s.upsertAgentCatalog(ctx, []string{""}, nil)
	if _, err := s.st.AgentImages().Get(ctx, ""); err == nil {
		t.Fatalf("empty image name should be skipped")
	}
}
```

- [ ] **Step 3: Add the helper**

In `internal/cp/server.go`, add the method (place it near the other `Server` helpers, e.g. just below the `Register` handling or near `reconcileInventory`):

```go
// upsertAgentCatalog records each advertised image and the binaries it ships so the durable catalog
// (ListAgentImages + runnable validation) reflects what connected nodes can run. Idempotent across
// reconnects: created_at is preserved and the binary set is replaced. Errors are logged, not fatal —
// a catalog write must never break node registration.
func (s *Server) upsertAgentCatalog(ctx context.Context, images, binaries []string) {
	now := time.Now().Unix()
	for _, img := range images {
		if img == "" {
			continue
		}
		if err := s.st.WithTx(ctx, func(tx store.Store) error {
			return tx.AgentImages().Upsert(ctx, store.AgentImage{Image: img, CreatedAt: now}, binaries)
		}); err != nil {
			log.Printf("register: upsert agent image %q: %v", img, err)
		}
	}
}
```
