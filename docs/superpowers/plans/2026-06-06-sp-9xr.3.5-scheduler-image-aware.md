# sp-9xr.3.5 — Scheduler: image-aware placement + thread selection into StartSpawn Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** When a spawn has an image selection, place it only on a node that advertises that image, and carry `image`/`runnable_id`/`mode` to the node in `StartSpawn`. Legacy spawns (empty selection) place anywhere, exactly as today.

**Architecture:** Add `Image` to `registry.Placement` and filter in `PickFor`. Add `runnable, mode` params to `scheduler.Provision` and set `Image`/`RunnableId`/`Mode` on the `StartSpawn` it sends (image read from `placement.Image`). Wire all three `Provision` callers — `provisionSpawn` (create), and the resume/recreate paths — to set `placement.Image = sp.Image` and pass `sp.RunnableID`/`sp.Mode`, so a resumed/recreated spawn keeps its agent. The node honoring these is `sp-9xr.3.6`.

**Tech Stack:** Go (`spawnery`), `internal/cp/scheduler`, `internal/cp/registry`, Connect node stream, stdlib `testing`.

**Run Go with:** `export PATH="/var/home/alepar/sdk/go1.26.0/bin:$PATH"`.

func TestCreateSpawnImageAwarePlacement(t *testing.T) {
	s, reg, _ := newTestServer(t)
	ctx := auth.WithOwner(context.Background(), "alice")

	// Catalog (for validation) + a node that advertises the image (for placement).
	s.upsertAgentCatalog(context.Background(), []string{"img:1"}, []string{"goose"})
	sender := &capSender{}
	reg.Add(&registry.Node{ID: "n1", Sender: sender, Max: 1, Free: 1, Images: []string{"img:1"}})

	go func() {
		for {
			if st := sender.firstStart(); st != nil {
				s.sched.OnStatus(st.GetSpawnId(), nodev1.SpawnPhase_ACTIVE)
				return
			}
		}
	}()

	resp, err := s.CreateSpawn(ctx, connect.NewRequest(&cpv1.CreateSpawnRequest{
		AppId: "secret-app", Model: "m", Image: "img:1", RunnableId: "goose-acp",
	}))
	if err != nil {
		t.Fatalf("CreateSpawn: %v", err)
	}
	waitActive(t, s, resp.Msg.SpawnId)

	st := sender.firstStart()
	if st == nil {
		t.Fatal("no StartSpawn sent")
	}
	if st.GetImage() != "img:1" || st.GetRunnableId() != "goose-acp" || st.GetMode() != "acp" {
		t.Fatalf("StartSpawn missing selection: image=%q runnable=%q mode=%q",
			st.GetImage(), st.GetRunnableId(), st.GetMode())
	}
}
