# sp-9xr.1 ÔÇö `agent_images` Table + DAO Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** Persist a registry of supported agent **images** (keyed by FQDN) and the list of agent **binaries** each image ships, with a CRUD DAO in the CP store.

**Architecture:** Two new tables ÔÇö `agent_images` (one row per FQDN) and `agent_image_binaries` (child rows, one per binary) ÔÇö added via a goose migration, modelled with Bun structs, and accessed through a new `AgentImageRepo` on the existing `store.Store` facade. This mirrors the existing `Spawn` + `spawn_mounts` parent/child pattern exactly. It is **pure storage**: it does not validate binary names against the registry and does not touch CreateSpawn ÔÇö that wiring is `sp-9xr.3`. This task is fully independent of `sp-9xr.2`.

**Tech Stack:** Go (module `spawnery`), Bun ORM (`github.com/uptrace/bun`) over SQLite (embedded, prod + tests) and Postgres, goose migrations (`github.com/pressly/goose/v3`), stdlib `testing`.

**Spec:** `docs/superpowers/specs/2026-06-06-tmux-terminal-mode-design.md` (┬ž"The capability taxonomy" ÔÇö "Image ÔÇö a Postgres `agent_images` row: FQDN + author-declared list of binaries").

### Conventions to copy (verified in tree)
- Repo shape: `type xRepo struct{ db bun.IDB }` + `var _ XRepo = (*xRepo)(nil)` ÔÇö `internal/cp/store/owners.go`.
- `ErrNotFound` on missing reads via `errors.Is(err, sql.ErrNoRows)` ÔÇö `owners.go:Get`.
- Caller supplies timestamps in the struct (no time generation in the repo) ÔÇö `Owner.CreatedAt`, `Spawn.CreatedAt`.
- Multi-statement writes run on `r.db` and rely on the caller's `WithTx` for atomicity ÔÇö `apps.go:UpsertVersion`.
- Test store: `store.NewTestStore(t)` (in-memory SQLite, migrated, FK-enforced) + `inTx(t, st, fn)` ÔÇö `internal/cp/store/testing.go`.
- Migration style: parallel `sqlite/` + `pg/` files, `-- +goose Up` / `-- +goose Down` ÔÇö `migrations/sqlite/0001_init.sql`.

### Schema note
The binary list is stored as **child rows** (not a delimited/JSON column), matching the `spawn_mounts` relational pattern. The child column is named `binary_name` (not `binary`) because `BINARY` is a reserved/keyword token in both SQLite and Postgres.

```go
func (r *agentImageRepo) Upsert(ctx context.Context, img AgentImage, binaries []string) error {
	// Keep created_at on conflict (DO NOTHING), then replace the binary set.
	if _, err := r.db.NewInsert().Model(&img).
		On("CONFLICT (image) DO NOTHING").
		Exec(ctx); err != nil {
		return err
	}
	if _, err := r.db.NewDelete().Model((*AgentImageBinary)(nil)).
		Where("image = ?", img.Image).Exec(ctx); err != nil {
		return err
	}
	for _, b := range binaries {
		row := AgentImageBinary{Image: img.Image, Binary: b}
		if _, err := r.db.NewInsert().Model(&row).Exec(ctx); err != nil {
			return err
		}
	}
	return nil
}

func (r *agentImageRepo) Get(ctx context.Context, image string) (AgentImage, error) {
	var img AgentImage
	err := r.db.NewSelect().Model(&img).Where("image = ?", image).Scan(ctx)
	if errors.Is(err, sql.ErrNoRows) {
		return AgentImage{}, ErrNotFound
	}
	return img, err
}

func (r *agentImageRepo) Binaries(ctx context.Context, image string) ([]string, error) {
	var rows []AgentImageBinary
	if err := r.db.NewSelect().Model(&rows).
		Where("image = ?", image).Order("binary_name ASC").Scan(ctx); err != nil {
		return nil, err
	}
	out := make([]string, len(rows))
	for i, row := range rows {
		out[i] = row.Binary
	}
	return out, nil
}
```
